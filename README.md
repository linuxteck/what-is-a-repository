## ⚡ 1-Minute Understanding

If you remember just this:

- Repository = software store for Linux  
- Contains packages, updates, and dependencies  
- Managed via package managers (`apt`, `dnf`, `pacman`)  

💡 It’s like an app store — but for your system.

## 🔄 Types of Repositories

| Type | Description |
|------|------------|
| Official | Maintained by distro (stable & secure) |
| Third-Party | External sources (extra software) |
| PPA (Ubuntu) | Personal package archives |
| Local | Stored on your system |

## 🚀 Common Commands (Copy-Paste Ready)

```bash
# Update repository list
sudo apt update

# Install a package
sudo apt install nginx

# Search for a package
apt search nginx

# Show repo sources
cat /etc/apt/sources.list
```
